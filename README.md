# Система прогнозирования продаж на основе исторических данных

Это проект по созданию системы прогнозирования продаж с использованием машинного обучения. Программа анализирует исторические данные о продажах и строит прогноз на будущий период (год) для нескольких регионов.

## 📊 Функциональность

Программа выполняет полный цикл работы с данными: от загрузки и подготовки до построения моделей, оценки качества и визуализации результатов.

### Основные возможности:

1.  **Загрузка и подготовка данных**
    *   Импорт данных из Excel-файла (`Исходные данные.xlsx`).
    *   Работа с данными по регионам: "Регион 1", "Регион 14", "Регион 16".
    *   Временной диапазон данных: с 2022-01-01 по 2025-09-30.

2.  **Подготовка признаков (Feature Engineering)**
    *   Создание временных признаков: год, месяц, день недели, квартал.
    *   Генерация циклических признаков (синус/косинус) для учета сезонности.
    *   Отметка российских праздников (Новый год, 8 Марта и др.).

3.  **Построение моделей**
    *  Учитывает годовую и недельную сезонность, а также праздничные дни.
    *  Использует лаговые признаки (сдвиги продаж за прошлые периоды: 7, 30, 90 дней и т.д.) для прогнозирования.

4.  **Оценка качества**
    *   Расчет метрик MAE (средняя абсолютная ошибка) и MAPE (средняя абсолютная процентная ошибка).
    *   Сравнение прогнозов с фактическими данными за 2025 год.

5.  **Прогнозирование**
    *   Генерация прогноза на весь 2026 год для каждого региона.
    *   Создание доверительных интервалов для прогнозов.

6.  **Визуализация**
    *   Графики временных рядов продаж.
    *   Анализ сезонных паттернов (средние продажи по месяцам).
    *   Сравнительные графики прогнозов по регионам.

### 📁 Результат работы

По завершении работы программа формирует Excel-файл `forecast_2026_all_regions.xlsx` со следующими листами:

*   **Детальный_прогноз:** Ежедневный прогноз продаж для всех регионов.
*   **Сводка_по_месяцам:** Прогноз в разрезе месяцев и регионов.
*   **Годовая_сводка:** Итоговые суммы продаж по регионам за год.
*   **Качество_моделей:** Показатели точности (MAPE) для оценки надежности прогноза.

## 📈 Практическое применение

Расчёты выполняются на **период один год** и могут быть использованы для:

| Область применения | Описание |
|-------------------|----------|
| **Прогнозирование производства** | Планирование объёмов выпуска продукции на основе ожидаемого спроса по месяцам |
| **Управление запасами** | Оптимизация складских остатков, предотвращение дефицита или избытка товаров |
| **Планирование закупок** | Расчёт потребности в сырьё и материалах с учётом сезонных колебаний продаж |
| **Бюджетирование** | Формирование реалистичного бюджета продаж и дохода на предстоящий год |
| **Логистика** | Планирование поставок и распределение товаров по регионам |

Прогноз на год вперёд позволяет предприятиям **заблаговременно подготовиться к пиковым нагрузкам**, скорректировать производственные мощности и оптимизировать цепочки поставок ещё до наступления высокого сезона.
